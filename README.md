# Waste2Cash

## Use Cases
- Information about different wastes and how they can be recycled
- Recyclable market
    - Sell your wastes
    - Buy others waste to recycle
- Waste Pickup
    - Become a waste pickerx    
    - Hire a waste picker
- Recycling machines market
- Local recycle centers
- Report unwanted trash
- Give Feedback

## Entities
- Waste
    - Id
    - Name
    - Type
    - Price
    - Order type (buy/sell)
    - Owner
    - Location
